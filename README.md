## ğŸ“… Managing Events (`events.csv`)

The portal automatically categorizes events based on the **Type** column:

1. **Special Events:** Set Type to `Yearly`. If the date is in the future, it automatically appears in the "Upcoming Specials" sidebar formatted as `Sunday 15/02/2026`.
2. **Regular Activities:** Set Type to `Monthly`. These appear in the "Monthly Regulars" section.
3. **Media Expansion:** Add a folder name (e.g., `shiva26`) in the `MediaFolder` column to enable the "View Media" expansion button.

## ğŸ”’ Security & Privacy Features

To protect the Trust from spam and hacking, the following are active:

* **Rate Limiting:** Users can only submit the Contact Form once every 60 seconds (stored in `localStorage`).
* **Sanitization:** JavaScript blocks malicious characters (`<`, `>`, `script`) from form inputs.
* **Honeypot:** A hidden field catches 99% of automated spam bots.
* **Privacy Agreement:** A mandatory checkbox ensures users consent to data usage for Samithi purposes only.
